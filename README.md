这是大二的实训项目 
用server+bootstrap+jquery+Mysql用传统的mvc模式来编写
MVC模式（Model–view–controller）是软件工程中的
一种软件架构模式，把软件系统分为三个基本部分：
模型（Model）、视图（View）和控制器（Controller）
